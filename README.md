# Electron Start

パッと作りたいときに使う予定。

## Usage

* `main/` メインプロセス用
* `renderer/` レンダラープロセス用
* `src/` レンダラーで使うものの事前ファイル
* `dist/` ビルドした App が入る
* `build/` ビルドに使うファイル
